# Sudoku Master 🦞💼

这是一个基于规格驱动开发（Spec-driven Development）理念构建的数独求解 Web 应用。项目最初由主人提供核心算法，并由龙管家（OpenClaw）配合 OpenSpec 与 OpenCode 自动化重构而成。

## 🌟 项目亮点

- **智能求解**：结合了逻辑推理（Junior Step）与递归回溯算法（Senior Backtracking），可秒杀“世界最难”数独。
- **现代化 Web 界面**：响应式 9x9 输入网格，支持实时输入与一键求解。
- **规格驱动**：项目包含完整的 OpenSpec 规格文档，记录了从需求到设计的每一个决策。

## 📂 文件说明

- `app.py`: 基于 Flask 的后端服务，负责处理 Web 请求并调度求解逻辑。
- `legacy_logic.py`: 核心算法库。包含数独校验、逻辑推理、候选数计算及递归回溯求解器。
- `templates/index.html`: 前端界面，负责 9x9 网格的渲染及与后端的异步交互。
- `openspec/`: 包含项目的技术规格、设计文档及变更历史。

## 🚀 快速开始

### 1. 安装依赖
确保您的环境中已安装 Python 3，然后运行：
```bash
pip install Flask numpy
```

### 2. 启动服务
在项目根目录下执行：
```bash
python app.py
```

### 3. 访问应用
启动后，在浏览器中打开以下地址：
`http://127.0.0.1:5000`
*(如果是远程服务器，请将 127.0.0.1 替换为您的服务器公网 IP)*

## 💡 使用说明
1. 在网页的 9x9 网格中填入已知的数独数字。
2. 留空的方格代表待求解的数字。
3. 点击底部的 **“求解”** 按钮，系统将自动计算并填充所有缺失数字。
